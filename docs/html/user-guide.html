Introduzione

FANTACONTRATTI e' una piattaforma di fantacalcio dinastico con sistema di contratti realistici. A differenza del fantacalcio tradizionale, le rose si mantengono nel tempo e ogni giocatore ha un contratto con ingaggio, durata e clausola rescissoria.

Caratteristiche Principali

  • Rose pluriennali che si mantengono tra le stagioni
  • Sistema contratti con ingaggio, durata e clausole
  • 5 fasi di mercato dinamiche
  • Meccanismo RUBATA per il riequilibrio competitivo
  • Aste in tempo reale via WebSocket
  • Storico completo di tutti i movimenti

Super Admin Piattaforma

Il Super Admin gestisce la piattaforma a livello globale. Non partecipa alle leghe come manager.

Dashboard Super Admin

Dopo il login, vieni reindirizzato automaticamente a /superadmin. La dashboard presenta 4 tab:

Upload Quotazioni Giocatori Leghe Utenti

Upload Quotazioni

Gestione del database giocatori tramite upload di file Excel.

1

Seleziona File XLSX

Clicca su "Upload Quotazioni" e seleziona il file Excel con le quotazioni ufficiali.

2

Conferma Upload

3

Verifica Report

Panoramica e gestione di tutti gli utenti registrati.

La promozione a Super Admin da' accesso completo alla piattaforma. Usa questa funzione con cautela.

Gestione Fasi di Mercato

Tipi di Sessione

  • Primo Mercato: Prima sessione con asta libera per costruire le rose
  • Mercato Ricorrente: Sessioni successive (Contratti, Scambi, Rubata, Svincolati, Premi)

Azioni Disponibili

  • Crea nuova sessione di mercato
  • Imposta ordine turni dei manager
  • Configura timer asta
  • Avanza o retrocedi tra le fasi
  • Reset Primo Mercato (se necessario)

Pannello Amministrazione

Mercato Membri Premi Ricorsi Inviti Audit

Membri

  • Approva richieste pendenti
  • Sospendi membri attivi
  • Visualizza rosa di ogni manager
  • Monitora budget disponibili

Premi

Durante la fase PREMI:

  1. Imposta il re-incremento base
  2. Crea categorie premio personalizzate
  3. Assegna premi per categoria
  4. Finalizza la fase

Ricorsi

Gestione delle contestazioni durante le aste:

  • Rivedi storico offerte
  • Accetta o rifiuta con motivazione
  • Se accettato, l'asta viene annullata

Audit Log

Registro completo di tutte le azioni: timestamp, utente, azione, valori precedenti/nuovi.

Direttore Generale Squadra

Rosa (Roster)

Visualizzazione

  • Giocatori ordinati per ruolo (P, D, C, A)
  • Dettagli contratto per ogni giocatore
  • Quotazione attuale e squadra reale

Dettagli Contratto

  • Ingaggio: Costo per semestre
  • Durata: Semestri rimanenti (1-4)
  • Clausola Rescissoria: Prezzo per la rubata

Slot Disponibili

Monitora gli slot per ruolo: Portieri, Difensori, Centrocampisti, Attaccanti.

Sistema Aste

Primo Mercato

L'asta iniziale per costruire la rosa:

  1. A turno, ogni manager nomina un giocatore
  2. Parte l'asta con timer
  3. Tutti possono rilanciare
  4. Il timer si resetta ad ogni rilancio
  5. Quando scade, il miglior offerente vince

Profezie

Puoi lasciare una "profezia" sul giocatore: un commento ironico o serio che sara' visibile a tutti dopo l'asta. Crea engagement nella lega!

Fase Rubata

Il meccanismo di riequilibrio competitivo:

  1. Ogni manager "espone" i propri giocatori
  2. Gli altri possono "rubarli" pagando clausola + ingaggio
  3. Se qualcuno paga, il giocatore cambia squadra
  4. Impossibile rifiutare!

Strategie Rubata

Imposta preferenze per ogni giocatore:

  • Watchlist: Giocatori da monitorare
  • Auto-Pass: Salta automaticamente
  • Max Bid: Offerta massima automatica
  • Priorita': Ordine di preferenza
  • Note private: Appunti strategici

Asta Svincolati

Acquisto di giocatori senza contratto (free agent):

  • Nomina un giocatore svincolato
  • Parte l'asta libera
  • Il prezzo base e' la quotazione
  • Vince il miglior offerente

Gestione Contratti

Durante la fase CONTRATTI puoi rinnovare i contratti dei tuoi giocatori:

  • Modifica l'ingaggio (entro limiti)
  • Modifica la durata (1-4 semestri)
  • Segna per lo svincolo (taglio)
1

Seleziona Giocatore

Vai su Contratti e scegli il giocatore da rinnovare.

2

Imposta Parametri

Modifica ingaggio e durata, verifica l'anteprima costi.

3

Consolida

Quando hai finito con tutti i giocatori, clicca "Consolida".

Regole di Rinnovo

  • Aumenti/diminuzioni massime per parametro
  • Budget sufficiente richiesto
  • Clausola ricalcolata automaticamente

Scambi tra Manager

Proporre uno Scambio

  1. Vai su Scambi > Nuova Offerta
  2. Seleziona il manager destinatario
  3. Scegli giocatori da offrire e richiedere
  4. Aggiungi/richiedi budget (opzionale)
  5. Imposta la scadenza e invia

Ricevere un'Offerta

Quando ricevi un'offerta puoi:

  • Accetta: Lo scambio viene eseguito
  • Rifiuta: L'offerta viene chiusa
  • Controproposta: Modifica i termini

Storico e Movimenti

Registro completo di tutte le transazioni:

  • Acquisti Primo Mercato
  • Movimenti Rubata
  • Acquisti Svincolati
  • Scambi tra manager
  • Svincoli e rinnovi

Per ogni movimento: data, tipo, giocatore, manager coinvolti, prezzo, dettagli contratto.

Glossario

Ingaggio
Costo del giocatore per semestre
Clausola Rescissoria
Prezzo per "rubare" un giocatore nella fase Rubata
Rubata
Fase di aste forzate dove si possono acquisire giocatori altrui
Svincolato
Giocatore senza contratto, disponibile all'acquisto
Consolidamento
Conferma definitiva dei contratti rinnovati
Profezia
Commento/previsione lasciato su un'acquisizione

Calcolo Clausola Rescissoria

Clausola = Ingaggio x Moltiplicatore

Il moltiplicatore dipende dalla durata: contratti piu' lunghi = clausola piu' alta.
