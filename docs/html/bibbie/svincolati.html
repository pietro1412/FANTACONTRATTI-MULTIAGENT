Panoramica

👤 Cos'e l'Asta Svincolati

La Fase 5 del mercato ricorrente. I manager, a turno, possono acquistare giocatori svincolati (senza contratto attivo) tramite asta.

Precondizioni

  • ✓ Fase 4 (Rubata) completata
  • ✓ Giocatori svincolati disponibili (scadenza contratto, tagli, mai acquistati)
  • ✓ Almeno un manager con budget sufficiente
2

Ordine di Chiamata

🔄 Ordine Inverso alla Rubata

L'ordine di chiamata e inverso all'ordine della rubata. Chi era primo nella rubata (vantaggiato) sara ultimo negli svincolati (svantaggiato) e viceversa.

Ordine Rubata

  • 1 Manager A
  • 2 Manager B
  • 3 Manager C
  • 4 Manager D
  • 5 Manager E
  • 6 Manager F
⇄

Ordine Svincolati

  • 1 Manager F
  • 2 Manager E
  • 3 Manager D
  • 4 Manager C
  • 5 Manager B
  • 6 Manager A

Ogni manager, al proprio turno, sceglie un giocatore svincolato da mettere all'asta. Il turno ruota ciclicamente tra i manager ancora attivi.

3

Meccanica dell'Asta

⚽ Come il Primo Mercato

L'asta si svolge con le stesse meccaniche del Primo Mercato:

1
Base d'asta
  • 💰 Offerta libera: qualsiasi manager puo rilanciare
  • ⏱ Timer: configurabile, reset ad ogni offerta
  • 🏆 Vincitore: ultimo offerente allo scadere

Funzionalita Admin

✓ Conferma Chiamata

Quando un manager sceglie il giocatore, il sistema chiede conferma.

⏸ Pausa

Un manager puo richiedere uno stop temporaneo.

⚠ Rettifiche Admin
AzioneEffetto
Annulla fine astaRiprende dall'ultima offerta (timer riparte)
Annulla chiamataIl giocatore torna svincolato
Rettifica transazioneGiocatore torna libero, crediti restituiti
4

Contratto Default

📝 Stessa Formula del Primo Mercato
Ingaggio Math.max(1, Math.round(prezzo / 10))
Durata Sempre 3 semestri

Modifica Post-Acquisto

  • ↑ Solo in aumento (increase-only)
  • ↑ Ingaggio puo aumentare
  • → Durata puo aumentare (richiede prima aumento ingaggio)
  • 4 Max durata: 4 semestri

Vedi Bibbia CONTRATTI sezioni 3 e 4 per tabella fasce e regole complete.
5

Impatto Finanziario

Costo Acquisto Svincolato Costo totale = prezzo_asta + ingaggio_effettivo

Il manager deve avere bilancio sufficiente per ENTRAMBI prima di fare offerta.

Verifica Pre-Offerta bilancio ≥ offerta + ingaggio_default
💡
Offerta minima: Offerta 1 + ingaggio default 1 = serve bilancio ≥ 2.
📊 Esempio Completo: Bernabe
1
Manager A, bilancio 250
Vince asta svincolato Bernabe a 25
2
Contratto Default
Ingaggio = Math.max(1, Math.round(25/10)) = 3, durata 3
3
Costo con contratto default
25 + 3 = 28 → Bilancio = 250 − 28 = 222
4
Se modifica a 5x4
25 + 5 = 30 → Bilancio = 250 − 30 = 220

Rinuncia al Turno

✕ Rinuncia Definitiva

Un manager puo dichiarare di voler saltare il turno. Questa rinuncia e DEFINITIVA: il manager smette di partecipare all'asta svincolati anche per tutti i turni futuri.

  • ✕ Il manager esce dalla rotazione dei turni
  • ✕ Non puo piu chiamare giocatori
  • ? Puo ancora fare offerte su giocatori chiamati da altri? (da definire)
7

Fine Fase

La fase si conclude quando si verifica una delle seguenti condizioni:

CondizioneDescrizione
Niente svincolati Non ci sono piu giocatori svincolati disponibili
Nessun budget Nessun manager ha bilancio ≥ 2 (offerta minima 1 + ingaggio minimo 1)
Tutti rinunciano Tutti i manager ancora attivi hanno rinunciato al turno
Admin chiude L'admin puo dichiarare fine fase in qualsiasi momento
